Section 5 of 5: Phase 1 Scope

What we build first — the smallest useful version of Holly

Foundation and PRM Core are merged into a single Phase 1. Foundation alone is not useful to anyone — the first thing you ship should let you actually log a contact and an interaction.

Phase 1 — In Scope

Foundation
✓ Next.js 15 project, TypeScript, Tailwind CSS
✓ Prisma + PostgreSQL schema (all 6 tables)
✓ Auth.js — Google OAuth + email/password
✓ Redis session store
✓ Dockerfile + Coolify deployment config
✓ PWA manifest + service worker (offline shell)
PRM Core
✓ Contact list, search, create, edit, view
✓ Log Interaction — quick-entry modal (under 30s)
✓ Interaction history per contact
✓ Follow-up flags + pending follow-ups list
✓ Relationship health alerts (overdue contacts)
✓ Dashboard with summary stats
✓ Holly API — interactions, contacts, briefing, follow-ups
✓ API key management in Settings
✓ Offline entry with background sync
