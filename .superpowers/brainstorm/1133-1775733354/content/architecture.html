Key Decisions

ORM: Prisma
Type-safe schema, auto-generated types, migrations. Schema is the single source of truth for DB + TypeScript types.
Auth: NextAuth.js (Auth.js v5)
Google OAuth + credentials (email/password). Session stored in Redis. Holly gets a separate long-lived API key, not a session.
Holly API: Separate route group
/api/holly/* routes use API key middleware. Rate limited. Fully documented. Versioned from day one (/api/holly/v1/*).
Real-time: Server-Sent Events
SSE over WebSocket — simpler, works through proxies, sufficient for push notifications. No separate WebSocket server needed.
PWA: next-pwa
Service worker for offline caching. Background sync for entries made offline. Web push for reminders.
Deployment: Single Coolify service
Dockerfile builds the Next.js app. Coolify manages PostgreSQL + Redis as separate services. Environment variables injected at runtime.
